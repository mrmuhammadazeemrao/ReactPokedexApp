# Pokémon App

This React application retrieves and displays a list of Pokémon from a given API. Upon clicking on a Pokémon, it displays its details.

## How to Run

1. Clone the repository:

```bash
git clone <repository_url>
cd pokemon-app
```

## Install Dependencies

```npm install
```

## Start Application

```npm run dev
```

## Run Test Cases
```npm run jest
```
